Wanna test your short term memory ?


This game is all about remebering numbers in specific order.


How to Play?

  • Select any number from the grid of 0 to 9 numbers.
  • Computer chooses the second number and so on.
  • Each turn computer and user, repeat the whole number sequence from the beginning and add a new number is added to the sequence.

 

Note:

    Entering incorrect sequence ends the game.
    Every new number should be one between 1 and 9.
